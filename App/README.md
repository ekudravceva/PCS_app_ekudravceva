# Генератор лабиринтов и поиск пути

## Описание проекта

Консольное приложение на C++ для генерации лабиринтов и поиска кратчайшего пути. Проект демонстрирует работу классических алгоритмов на графах и предназначен для образовательных целей.

## Функциональные возможности

### Генерация лабиринта
- **DFS** (Depth-First Search) — генерация методом поиска в глубину
- **Kruskal** — генерация с использованием системы непересекающихся множеств (DSU)

### Поиск пути
- **BFS** (Breadth-First Search) — поиск кратчайшего пути в ширину
- **A\*** (A Star) — эвристический поиск оптимального пути

### Визуализация
- Консольное отображение лабиринта символами ASCII
- Пошаговый режим (ручное управление через Enter)
- Автоматический режим (анимация с заданной задержкой)

### Статистика
- Время генерации лабиринта (мс)
- Время поиска пути (мс)
- Длина найденного пути
- Количество посещённых ячеек

## Архитектура проекта

Проект реализован с использованием паттернов проектирования:

- **Стратегия** — взаимозаменяемые алгоритмы генерации и поиска
- **Команда** — обработка действий пользователя в консольном меню
- **Декоратор** — сбор статистики без изменения алгоритмов
- **Фабричный метод** — создание объектов алгоритмов

## Требования

- C++17 или новее
- CMake 3.10 или новее
- Компилятор: GCC, Clang, MSVC или Apple Clang (Xcode)

## Сборка и запуск

### macOS / Linux (терминал)

```bash
# Клонирование репозитория
git clone <url-репозитория>
cd MazeProject

# Сборка проекта
mkdir build
cd build
cmake ..
make

# Запуск
./MazeApp
```

### Windows (MinGW)

```bash
mkdir build
cd build
cmake .. -G "MinGW Makefiles"
mingw32-make
MazeApp.exe
```
 
### Для запуска:
``` 
cd build
./MazeApp
```
